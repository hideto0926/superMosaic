# superMosaic — Landing site

Marketing site for **superMosaic**, the on-device iOS app that auto-detects faces,
skin and sensitive areas and masks them with blur, mosaic, white-out or black-out.

- `index.html` — bilingual (EN / 日本語) landing page, feature-centric
- `privacy.html` — privacy policy
- `assets/` — app icon and screenshots

## Features highlighted
- Auto-detection of faces, exposed skin and sensitive areas
- Blur / mosaic / white-out / black-out / color effects
- Batch processing of multiple photos
- 4-in-1 mode: merge 4 photos while masking
- Strip top/bottom black bars from Instagram-style screenshots
- Erase magic, manual & pen mode
- 100% on-device — no photos leave the device

## Deploy (GitHub Pages)
Push to a repo and enable Pages on the root of the default branch.
`.nojekyll` is included so the `assets/` folder is served as-is.
